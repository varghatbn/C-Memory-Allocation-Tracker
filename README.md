# C-Memory-Allocation-Tracker
    C program designed to help understand memory Allocation/Deallocation in a low level programming environment.
    Allows users to allocate memory to student instances and keeps track of allocated memory and prints them 
    to the screen.
## Version 
    1.2.0
    
## Compilation Instructions
    complile in Ubuntu using: gcc -o a4 main.c stuDefs.h mhDefs.h
    launch after compilation with: ./a4
